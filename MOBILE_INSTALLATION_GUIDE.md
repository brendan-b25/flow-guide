## Installation Instructions

### iPhone/iPad (iOS/iPadOS)

#### Method 1: Safari (Recommended)

1. **Open Safari** and navigate to: `https://brendan-b25.github.io/flow-guide/`
2. **Tap the Share button** (square with arrow pointing up) at the bottom
3. **Scroll down** and tap "Add to Home Screen"
4. **Customize the name** if desired (default: "FlowGuide")
5. **Tap "Add"** in the top right corner
6. The app icon will appear on your home screen

#### Method 2: Install Prompt

1. When you visit the app, you may see an install prompt at the bottom
2. Tap **"Install Now"** on the prompt
3. Follow the on-screen instructions

### Android

#### Method 1: Chrome (Recommended)

1. **Open Chrome** and navigate to: `https://brendan-b25.github.io/flow-guide/`
2. **Tap the menu** (three dots) in the top right
3. **Tap "Install app"** or "Add to Home screen"
4. **Tap "Install"** in the popup
5. The app will be added to your home screen and app drawer

#### Method 2: Install Banner

1. Look for the install banner at the bottom of the screen
2. Tap **"Install"**
3. Follow the on-screen instructions

## Troubleshooting

### App Won't Install

**iOS:**
- Make sure you're using Safari (not Chrome or other browsers)
- Check iOS version (requires iOS 11.3+)
- Try clearing Safari cache

**Android:**
- Use Chrome browser
- Check Chrome version (requires Chrome 70+)
- Enable "Install from Unknown Sources" if needed

### Notifications Not Working

1. Check notification permissions in device settings
2. Ensure Do Not Disturb is disabled
3. Verify app has notification permission granted

### Offline Mode Issues

1. Visit key pages while online to cache them
2. Check storage space on your device
3. Clear app cache and reload: Settings â†’ Safari â†’ Clear History and Website Data

### Camera/Microphone Not Working

1. Go to device **Settings** â†’ **FlowGuide**
2. Enable Camera and Microphone permissions
3. Restart the app

### App Feels Slow

1. Clear app cache
2. Check internet connection
3. Update to latest version (reinstall from Safari)
4. Restart your device
